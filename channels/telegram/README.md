# Claudio - Canal Telegram

Bot de Telegram para interactuar con Claudio.

## Setup

```bash
cd channels/telegram
pip install -r requirements.txt
```

## Variables de Entorno

```bash
TELEGRAM_BOT_TOKEN=your_token
ANTHROPIC_API_KEY=your_key
```

## Ejecutar

```bash
python bot.py
```

## Funcionalidades

- Crear User Stories, Epics, Initiatives en ClickUp
- Consultar PRs en GitHub
- Buscar y crear documentos en Google Docs
- Enviar mensajes a Slack
