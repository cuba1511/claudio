# Workflows

Workflows que combinan múltiples MCPs para automatizar tareas comunes.

## Workflows Disponibles

| Workflow | MCPs | Descripción |
|----------|------|-------------|
| [Daily Standup](./daily-standup.md) | Slack + Google Docs | Crear notas de daily y compartir |
| [Create Initiative](./create-initiative.md) | ClickUp + Google Docs | Crear initiative con spec |
| [Sprint Report](./sprint-report.md) | ClickUp + Slack + Sheets | Reporte de sprint |

## Cómo crear un nuevo workflow

1. Identificar los MCPs necesarios
2. Definir los pasos en orden
3. Documentar inputs y outputs
4. Agregar ejemplos de uso
