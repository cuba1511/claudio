# Daily Standup Workflow

## MCPs Utilizados
- Google Docs (crear notas)
- Slack (compartir link)

## Pasos

1. **Crear documento** en Google Docs con template de daily
2. **Obtener link** del documento
3. **Enviar mensaje** a canal de Slack con el link

## Trigger

```
"Crea las notas para la daily de hoy"
```

## Template de Daily

```markdown
# Daily Standup - {fecha}

## Participantes
- 

## Updates

### [Nombre]
- **Ayer**: 
- **Hoy**: 
- **Blockers**: 

## Action Items
- [ ] 

## Notas
```

## Output

- Documento creado en Google Docs
- Mensaje enviado a #daily-standups con link
