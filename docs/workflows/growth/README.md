# Growth Workflows

Workflows for experiment lifecycle management via Notion.

## Workflows

| Workflow | MCPs | Trigger |
|----------|------|---------|
| [Create Experiment](./create-experiment.md) | Notion | "Crea un experimento para..." |
| [Experiment Review](./experiment-review.md) | Notion | "Prepárame la review de experimentos" |
| [Update Experiment](./update-experiment.md) | Notion | "Actualiza el experimento X con resultados" |
| [Conclude Experiment](./conclude-experiment.md) | Notion | "Cierra el experimento X" |

## Framework (always)

Every experiment must document:

1. **Problema** — quantified pain
2. **Hipótesis** — testable if/then/because
3. **Métricas** — primary, secondary, guardrails
4. **Metodología** — design, audience, duration, risks
5. **Sign-off** — stakeholders per `docs/integrations/notion/config.md`

## How to add a workflow

1. Identify MCPs (usually Notion only)
2. Define trigger phrases
3. Document steps, inputs, outputs
4. Link to `docs/integrations/notion/templates/experiment.md`
